# QA Playground

A playground for experimenting with QA automation tools and practices.

## Structure
- **backend/** → Node.js + Express API
- **frontend/** → Angular client
- **tests/** → Postman, automation, and performance tests

## Quick Start
1. Start backend: `npm start` (inside `backend`)
2. Run Postman tests:  
   ```bash
   newman run tests/postman/qa-playground.postman_collection.json
